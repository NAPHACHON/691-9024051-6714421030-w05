# สรุปงานและส่วนที่แก้ไขในโปรเจกต์ ebd-w05

### 1. ระบบจัดการ Tag (CRUD)
* **Controller:** สร้าง `TagController` เพื่อควบคุมการทำงาน (แสดง, เพิ่ม, แก้ไข, ลบ)
* **Model:** สร้าง `Tag.php` พร้อมตั้งค่า `$fillable = ['tag_name']` ให้บันทึกข้อมูลได้
* **Views:** สร้างหน้าแสดงผล (`index`), ฟอร์มเพิ่ม (`create`), ฟอร์มแก้ไข (`edit`), และหน้าแสดงรายละเอียด (`show`)
* **ปรับแต่งหน้าเว็บ:** เอาปุ่ม Detail ออก, แก้ไขปุ่ม Edit เป็นสีเหลือง และปุ่ม Delete เป็นสีแดง

### 2. เพิ่มระบบ CRUD ให้ตารางอื่นๆ ทั้งหมด (Title, Staff, Activity, Organization, Community)
* **Controllers:** เพิ่มฟังก์ชัน `index`, `create`, `store`, `edit`, `update`, `destroy` ลงในคอนโทรลเลอร์ทุกตาราง
* **Models:** ตั้งค่า custom table name และ `$fillable` ให้กับโมเดลที่ต้องการ
* **Views:** 
  - สร้างหน้าจอและฟอร์ม Add / Edit ครบทุกตาราง
  - เพิ่มปุ่ม Add, Edit (สีเหลือง), Delete (สีแดง) ลงในหน้าแสดงข้อมูลของทุกตาราง
  - สำหรับหน้า **Staff** และ **Activity** มีระบบ Dropdown ให้เลือกคำนำหน้า, หน่วยงาน และ ชุมชน อย่างถูกต้อง

### 3. ย้ายไปใช้ฐานข้อมูล MySQL
* ปรับตั้งค่าที่ไฟล์ `.env` เพื่อเปลี่ยนจาก SQLite ไปใช้งาน MySQL ชื่อฐานข้อมูล `ebd-w05`
* สร้างตารางและใส่ข้อมูลตัวอย่าง (Seeder) ของทุกๆ ตารางลงใน MySQL เรียบร้อยแล้ว

### 4. แก้ไขบั๊กบน Windows
* แก้ไข `composer.json` เอาคำสั่ง `php artisan pail` ออก เพื่อป้องกันระบบพังตอนรันบน Windows (ส่งผลให้สามารถรันคำสั่ง `composer run dev` ได้ปกติ)
* แก้ไขหน้าเว็บจากที่ขึ้นชื่อสัปดาห์เก่า `ebd-w04` ให้เปลี่ยนเป็น `ebd-w05` ทุกหน้าแล้ว
